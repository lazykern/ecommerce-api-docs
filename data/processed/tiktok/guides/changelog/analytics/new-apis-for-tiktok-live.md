# Summary
We've received a lot of requests to add TikTok LIVE analytics to our ever-growing TikTok Shop analytics APIs. We're happy to announce that we've recently just launched two new endpoints to address this demand!
# Impact
|  |  |
| --- | --- |
| Impacted market(s) | * All markets - local and cross-border |
| Impacted version(s) | * 202309 (and later) |
# Changes
## New analytics APIs
The following endpoints and have been newly launched:
| **Endpoint** | **Description** |
| --- | --- |
| [GET Shop Live Performance Overview](https://partner.tiktokshop.com/docv2/page/get-shop-live-performance-overview-202508) | Returns overall performance metrics for all LIVE stream sessions under a certain shop. |
| [GET Shop Live Performance List](https://partner.tiktokshop.com/docv2/page/get-shop-live-performance-list-202508) | Returns a list of LIVE stream sessions and associated metrics for a shop. |

For those familiar with our video performance endpoints, namely [GET Shop Video Performance Overview](https://partner.tiktokshop.com/docv2/page/get-shop-video-performance-overview-202409) and [GET Shop Video Performance List](https://partner.tiktokshop.com/docv2/page/get-shop-video-performance-list-202409), these new LIVE endpoints have been structured to mirror those two APIs.
