# Summary
Today, we've opened up all our analytics APIs to be available globally! Endpoints that were previously exclusive to the US, UK, and SEA can now be used to return performance metrics for any shop, regardless of market. 
# Impact
|  |  |
| --- | --- |
| Impacted market(s) | * All markets - Local and cross-border |
| Impacted version(s) | * 202405 (and later) |
# What's new?
All existing Analytics APIs are now available in all markets. These include:
– [GET Shop Performance](https://partner.tiktokshop.com/docv2/page/get-shop-performance-202405)
– [GET Shop Product Performance](https://partner.tiktokshop.com/docv2/page/get-shop-product-performance-202405)
– [GET Shop Product Performance List](https://partner.tiktokshop.com/docv2/page/get-shop-product-performance-list-202405)

– [GET Shop SKU Performance List](https://partner.tiktokshop.com/docv2/page/get-shop-sku-performance-list-202406)
