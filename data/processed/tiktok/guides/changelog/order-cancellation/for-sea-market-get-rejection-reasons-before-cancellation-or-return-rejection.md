Since cancellation or return rejection reasons change sometimes, please call [Get Rejection Reasons](https://partner.tiktokshop.com/docv2/page/get-reject-reasons-202309#Back%20To%20Top) and show seller the usable rejection reason list if seller wants to reject buyer's cancellation or return request. You can get this list from the "reasons" parameter in this API's response payload. It will be effective after Sep. 24, 2025. 
By the way, you can get the newest [return reasons](https://partner.tiktokshop.com/docv2/page/return-reasons), [refund reasons](https://partner.tiktokshop.com/docv2/page/refund-reasons) and [cancel reasons](https://partner.tiktokshop.com/docv2/page/cancel-reasons) from links as below:

* Return reasons: https://partner.tiktokshop.com/docv2/page/return-reasons
* Refund reasons: https://partner.tiktokshop.com/docv2/page/refund-reasons
* Cancel reasons: https://partner.tiktokshop.com/docv2/page/cancel-reasons

# Which markets are affected?
The updates of the requirements apply to the Local to Local sellers for Singapore, Philippines, Vietnam, Thailand, Malaysia and Indonesia markets.
# Who is affected?
Developers with applications that use Cancellation, Return or Refund related APIs.
# Which version is applicable?
The updates of the requirements apply to all versions of the related API
# What action is required?
Developers need to check whether to use these APIs. Please call [Get Rejection Reasons](https://partner.tiktokshop.com/docv2/page/get-reject-reasons-202309#Back%20To%20Top) and show seller the usable rejection reason list if seller wants to reject buyer's cancellation or return request.
