# What is changing?
From August 31 2025, three seller initiates cancellation reasons will be changed.

1. "seller_cancel_paid_reason_address_not_deliver" will be removed in Indonesia, Philippines, Vietnam, Malaysia and Thailand markets.
2. "seller_cancel_unable_to_deliver_to_buyer_address" will be removed in Singapore market.
3. "seller_cancel_reason_out_of_stock" will be adjusted from "Buyer Fault" to "Seller Fault" in Indonesia market.

# Which markets are affected?
The updates of the requirements apply to the Local to Local sellers for Singapore, Philippines, Vietnam, Thailand, Malaysia and Indonesia markets.
# Who is affected?
Developers with applications that use Cancel Order.
# Which version is applicable?
The updates of the requirements apply to all versions of the related APIs
# What action is required?
Developers need to check whether to use these APIs and handle the related error messages.
