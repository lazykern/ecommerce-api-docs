## Summary
The Listing Schema endpoint will be sunset on **August 31, 2024**.
## Affected Markets and API Versions
This change is for all markets and all API versions.
## Required Actions
The capabilities of the Listing Schema endpoint can be fulfilled by the [Get Attributes](get-attributes) and [Get Category Rules](get-category-rules) endpoints, which will remain available. Please transition to using these two endpoints and discontinue using the Listing Schema endpoint in your app by August 31, 2024.
