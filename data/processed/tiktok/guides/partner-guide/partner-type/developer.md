# What is a Developer?
Using the TikTok Shop APIs, developers build applications that offer sellers solutions to automate, manage and grow their businesses on TikTok Shop.
Via the TikTok Shop Partner Center, we provide a comprehensive set of technical tools, resources, and APIs that allow developers to integrate directly with TikTok Shop.
Our APIs span the entirety of the seller operation process, allowing developers to create integrations that can streamline merchant workflows and improve customer experiences.

# Developer Type
There are three types of developers: seller developer, app developer, and system integrators. See below for more information.
| Developer type | Definition | App type |
| --- | --- | --- |
| Seller Developer <br> (TikTok Shop Seller) | Seller developers are developers who directly integrate TikTok Shop with their system. Typically, these are sellers who need custom solutions to support their own operations on TikTok Shop. **This is a direct 1:1 authorization for the bound TikTok Shop Seller Account.** | Custom App |
| System Integrator | Technology Service Providers who are authorized by certain sellers to create a custom integration | Custom App <br> Public App |
| App Developer | App developers are developers who build apps to support seller operations on Tiktok Shop. | Custom App <br> Public App |
