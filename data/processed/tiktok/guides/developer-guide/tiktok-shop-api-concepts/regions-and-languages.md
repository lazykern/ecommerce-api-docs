TikTok Shop supports retrieval of text data for different regions in a variety of languages through codes in the [IETF BCP 47](https://www.unicode.org/reports/tr35/#Unicode_locale_identifier) standard.
# Supported languages
You can pass the following locale codes in APIs that support localization to obtain text data in the corresponding language.
| **Language** | **Locale codes** |
| --- | --- |
| English | en-GB, en-IE, en-PH, en-SG, en-US |
| French | fr-FR |
| German | de-DE |
| Indonesian | id-ID |
| Italian | it-IT |
| Japanese | ja-JP |
| Malay | ms-MY |
| Portuguese | pt-BR |
| Spanish | es-ES, es-MX |
| Thai | th-TH |
| Vietnamese | vi-VN |
