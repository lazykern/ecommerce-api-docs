Before you begin this tutorial to make your first TikTok Shop API call, you need to have a [TikTok Shop Partner Center](https://partner-sso.tiktok.com/account/login) account, and that account must be [registered as a developer](developer-onboarding).
If you've completed those steps, the tutorial to make your first TikTok Shop API call has the following steps:

1. [Create a TikTok Shop App (OAuth client)](create-tts-app-oauth-client)
2. [Create a test seller account](create-test-seller-account)
3. [Generate a test access token](generate-test-access-token)
4. [Create a hash to sign your test API call](create-hash-to-sign-your-test-api-call)
5. [Call Get Authorized Shops](call-get-authorized-shops)
