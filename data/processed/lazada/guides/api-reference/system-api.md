The table below lists APIs that are provided by the Open Platform.

**API name** | **Description**  
---|---  
[generate access_token](<https://open.lazada.com/apps/doc/api?path=%2Fauth%2Ftoken%2Fcreate>) | Get access token with the authorization code that is received from the callback URL when sellers complete APP authorization.   
[refresh access_token](<https://open.lazada.com/apps/doc/api?path=%2Fauth%2Ftoken%2Frefresh>) | Refresh the access token before it expires.
