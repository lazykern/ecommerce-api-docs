The table below lists APIs that can be used for seller information (email address) management.

API name | Description
---|---
GetSeller | Get seller information by current seller ID.
GetSellerMetricsById | Get seller metrics data of the specific seller, like positive seller rating, ship on time rate and etc.
SellerUpdate | Update the email address of the seller who makes the call.
UserUpdate | Update the email address of a user under the seller account.
